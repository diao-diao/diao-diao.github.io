推荐软件

解压软件

市面上压缩软件有很多，例如Winrar，快压等

但是并不推荐,因为广告多，还经常弹出收费信息 这里推荐7-zip

下载地址：https://www.7-zip.org/

该软件能够解压所有的压缩包(zip,rar,tar)，它还有特有的格式.7z

浏览器

浏览器并不推荐Windows自带的IE,总体使用方便程度上并不高

这里推荐Chrome和Firefox

360浏览器和搜狗等浏览器的内核均是使用
Chrome，但是广告较多，建议直接使用Chrome

Chrome:https://www.google.cn/chrome/ Firefox:http://www.firefox.com.cn/download

杀毒软件

杀毒软件是必要的，Windows10自带有杀毒软件，是目前测评第一的杀毒软件，但是它面对一些破解 软件或者文件的时候，会出现误报，导致繁琐问题

这里推荐火绒 https://www.huorong.cn/person5.html

免费，无广告，占用内存最低，病毒库平均2-3天自动更新一次，比较适合

使用360等其他软件也可以

下载软件

由于计算机的一些局限性，有一部分文件和软件都在外网上，而天朝有特色，所以会导致访问速度慢， 甚至无法访问的情况出现

这里迅雷和百度云是必要的，当面对浏览器下载速度只有kb时，采用迅雷和百度云可以解决大部分问

论坛/网站

我个人主要常用的论坛有 CSDN，简书和Github

善用CSDN和简书可以帮助你解决大部分的电脑问题，培养自主学习和动手能力

Github是一个代码托管的网站，你可以在上面参考大神的代码去提高你的代码健壮度

编译器

推荐CodeBlocks和DEV

个人喜欢CodeBlocks，容易上手

DEV是日后工作常用和必备的C/C++编译器

CodeBlocks会上传到群里

编辑器

编辑器有很多，注意与编译器进行区分，你可以把记事本功能理解为Windows自带的编辑器

常见的编辑器有VS Code ; Sublime text ; Notepad++

VS Code: https://code.visualstudio.com/Download

Sublime text: http://www.sublimetext.com/3

Notepad++ : https://notepad-plus-plus.org/downloads/

建议下载VS Code和Notepad++

前者编写HTML代码 配合插件更加方便 由微软开发

Notepad++是很强力的一款编辑器，它可以打开几乎所有文件，无论是二进制文件还是常见的word

Sublime text也同样优秀，但是操作比较繁琐，再配合上天朝特色，会自闭

远控软件

推荐TeamViewer

但是注意不要连续与同一个人通信，一次通信时间不要过长

TeamViewer是收费软件，但是针对个人是免费的，如果连续与同一个人通信或通信时间过长
可能会被判定为企业行为，就会被强制收费，无法使用

如果觉得麻烦还可以使用向日葵，但是流畅度相对于teamviewer较低

MarkDown

MarkDown的具体可以看看菜鸟

https://www.runoob.com/markdown/md-tutorial.html
https://sspai.com/post/54912

总而言之下来就是一款写文章写笔记都很好用的软件

推荐MarkDown2或Typora

MarkDown2可以实现边写边看

Typora快捷操作比较多，书写流畅

看个人意愿

MarkDown2： https://www.jianshu.com/p/1745a7f53404

Typora： https://www.typora.io/#download

建议写学习笔记，最好形成日记或周报的形式，有助于你的学习，使用md可以让你的文章更加清晰， 培养有关于目录结构的思维

到了工作岗位，普遍都需要使用md

重做系统

一般电脑买来之后都是默认为家庭版Win10系统，这里出一个教程帮助重做专业版以及日后重做系统

启动U盘

首先打开微软官网
https://www.microsoft.com/zh-cn/software-download/windows10

点击立即下载工具

打开工具之后，选择ISO镜像文件下载到电脑

接着使用UltraISO工具制作启动U盘

下载地址：https://cn.ultraiso.net/xiazai.html
使用方法：https://jingyan.baidu.com/article/154b46311befea28ca8f41ae.html

制作完毕之后，插上U盘就可以进行下一步操作了

系统安装

根据以下图文介绍即可

首先根据自身电脑的品牌进入BIOS模式
详情参考：https://jingyan.baidu.com/article/b24f6c82c11ad286bfe5da23.html

每个BIOS的不同，我的是最常见的面板，基本步骤都一样，都是找到启动选项


根据右侧的提示，使用F5或F6（F9或F10） 将U盘/USB硬盘的顺序提升到最高，UEFI和传统模式都提升至最高，保存并退出即可下一步


如果你前面都没有问题，那么就会来到这一步


这个地方先别急着现在安装，点击修复计算机


选择疑难解答


点击命令提示符



依次输入命令：diskpart->list disk->根据列表选择你的磁盘，每个人的磁盘不一样，灵活变通->select disk 磁盘号->clean->exit->exit

接着点关机，然后重启即可


接着回到这一步，点现在安装


点击我没有密钥


选择专业版


点击自定义安装


选择好磁盘之后，点击下一步


等待自动重启即可


接着回到BIOS，将系统内置硬盘提升到最高权限，并拔出U盘，保存并退出


完成之后，稍作等待就会看到这个界面


接下来按照自己配置即可


由于是正版系统，需要激活，激活软件会发到新生群里，直接打开激活即可
